## Run Locally

Clone the project

```bash
  git clone https://github.com/DihanVermeulen/dad-joke-generator
```

Go to the project directory

```bash
  cd dad-joke-generator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Using ESLint

